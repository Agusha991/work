Мое понимание о vue js

Данная технология очень удобна для верстки сайтов. Пользуется большим спросом в Китайских компаниях. Vue входит в ядро Laravel и PageKit. Это отличный фреймворк для разработки одностраничных приложений(SPA).

Реактивность

Реактивность это модели и при их изменении обновляется представление данных, и исходя из этого управление состоянием становиться довольно простым. Изменения отслеживаются с помощью первщания поля в пары геттер и сеттер для этого используется Object.defineProperty. С помощью их Vue отслеживает все изменения, к каждомы компоненту нужен экземпляр наблюдателя, он помечает все поля, которые затронула сортировка.

Шаблоны

Шаблон это валидный экземпляр HTML. Vue компилирует шаблоны в render функционального DOM. Vue знает минимальное количество компонентов и исходя из этого успользует меньше манипуляций для изменения DOM приложения. Так же есть виды интерполяции, текстовая интерполиция с использование Mustache. Есть сырой HTML для этого типа текст в {{}} подставляется как текст и для того чтобы вывелся нужно присвоить ему v-html="rawHtml". Атрибуты синтаксис двойных ковычек не работает в html для этого нужно использовать v-bind(это служит как проверки для выводы тэга). Дерективы v- принимает выражение js, если будет v-if='watch', тогда запись выведется если watch =true else null. Аргументы- могут принимать агрумент, отделен от директивы двоиточием. Модификаторы - особый префикс который после точки передает директиву которая привязана к какому то образу. Сокращение - это просто двоеточие которое сокращает v-bind.
